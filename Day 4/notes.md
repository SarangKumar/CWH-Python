## Day 4 - Our first program

#### How to run python script

- Save the file as `filename.py` (py is the extension for python files)
- Using cmd go to the path where the script is and run the below code

```shell
python filename.py
```
*or*
```shell
python3 filename.py
```

#### First code

```python
print('Hello World!')
```
